Documentation

Description

This module handles text ellipsis on multilines. You have to pass the number of lines and line-height (It is used to calculate the height). You can also define the tag (default is 'p').

    multiLineOptions: JSON.stringify({
        multiLine: 2,
        lineHeight: 1.5
    }),
    tag: 'h1',
    text: 'Lorem ipsum'
