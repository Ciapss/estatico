Documentation

Description

CSS based module to add corner ribbons. The ribbon expands and positions itself depending on the text length. It also can be multiline.

The module offers 2 tools:

  • The ribbon mixin
  • 3 variation classes

Usage

Ideally you should be defining the parent element where the module lives with:

.parent_module_or_wrapper {
    position: relative;
}

To position the ribbon in each corner you can either use the ribbon mixin or the var classes. the default position is the top-left corner.

Positioning the ribbon to the top-left corner

  1. Using the mixin:

     @include ribbon;

  2. Using the classes:

     <div class="mod_cornerribbon">
     <span>{{ribbonText}}</span>
 </div>

Positioning the ribbon to the top-right corner

  1. Using the mixin:

     @include ribbon($corner: top-right);

  2. Using the classes:

     <div class="mod_cornerribbon var_top-right">
     <span>{{ribbonText}}</span>
 </div>

Positioning the ribbon to the bottom-left corner

  1. Using the mixin:

     @include ribbon($corner: bottom-left);

  2. Using the classes:

     <div class="mod_cornerribbon var_bottom-left">
     <span>{{ribbonText}}</span>
 </div>

Positioning the ribbon to the bottom-right corner

  1. Using the mixin:

     @include ribbon($corner: bottom-right);

  2. Using the classes:

     <div class="mod_cornerribbon var_bottom-right">
     <span>{{ribbonText}}</span>
 </div>
